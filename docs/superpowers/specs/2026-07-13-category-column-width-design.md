# 참가종별 성별 열 동일 너비 설계

## 목표

관리자 대회 상세의 참가종별 표에서 남자와 여자 열을 항상 같은 너비로 표시한다. 카드 내부 문구는 단어 중간에서 끊지 않고, 필요한 경우 단어 단위로 여유 있게 줄바꿈한다.

## 결정

- 기존 table 구조를 유지하고 `table-fixed`와 `colgroup`으로 열 너비를 명시한다.
- 종별 열은 `7rem`으로 고정하고 남자·여자 열은 각각 `calc((100% - 7rem) / 2)`를 사용한다. 표의 최소 너비 `560px`에서는 각 성별 열이 카드 최소 너비 `7rem`보다 충분히 넓어야 한다.
- 카드의 한국어 문구에는 `word-break: keep-all`에 해당하는 `break-keep`을 적용해 음절 단위 줄바꿈을 막는다.
- 참가자·세부종목·점수 집계는 각 항목을 별도 `whitespace-nowrap` 요소로 감싸고, 바깥 컨테이너만 `flex-wrap`하도록 해 항목 전체가 다음 줄로 이동하게 한다.
- 짧은 상태 문구는 아이콘과 분리되지 않도록 `whitespace-nowrap`으로 유지한다.
- 기존 최소 표 너비와 가로 스크롤 동작은 유지한다.

## 대안

- CSS Grid 전환: 열 제어는 쉽지만 기존 표의 의미 구조와 스타일을 불필요하게 바꾼다.
- 성별 셀에 `w-1/2`만 지정: 자동 table layout의 콘텐츠 폭 계산 때문에 동일 너비가 보장되지 않는다.

## 검증

- 데스크톱과 좁은 화면에서 헤더와 각 행의 남자·여자 셀 `getBoundingClientRect().width` 차이가 1px 이내다.
- `미등록`/`사용 중`, 점수 항목 유무처럼 좌우 콘텐츠가 다른 조합에서도 두 열 너비가 동일하다.
- `사용 중지됨`, `참가자 N명`, `세부종목 N개`, `점수 N건`이 단어 또는 집계 항목 중간에서 끊기지 않는다.
- 모바일에서는 기존처럼 표 전체가 가로 스크롤된다.
- 타입 검사와 프로덕션 빌드가 통과한다.
